Welcome to UVic Courses API

This API provides access to Fall 2023 and Spring 2024 data from banner.uvic.ca, allowing you to integrate course data into your applications with ease.

How it was made:

The Courses API was built using Python with Django REST framework and PostgreSQL.

Purpose:

This API was created for the purpose of providing simple and easy access to course data in JavaScript.

Example endpoints:

            
 Section A03 for Math 109 in Fall 23 

            
                /api/202309/MATH/109/A03

                /sections/?refnum=12105
            
        
            
 Seng 265 sections in Spring 24 

            
                /api/202401/SENG/265/
            
        
            
 Third year math courses in Fall 23 

            
                /courses?subject=MATH&term=202309&level=3
            
        
            
 All spring courses in 2024 

            
                /api/202401                    (no pagination, slower response)
                
                /courses/?term=202401          (pagination, faster response)
            
        
            
 Meeting times on Monday and Thursday that start after 10AM and end before 1PM

            
                /meetingtimes?on=1001000&saft=1000&ebef=1300
            
        
            
 Meeting times that include Wednesday, start at 10AM, and end at 11:20 PM 

            
                /meetingtimes?days=0010000&start=1000&end=1120
            
        
            
 What rooms does Electrical and Computer Engineering use in Elliot building 

            
                /meetinglocations?bldg=Ell&subject=ECE
